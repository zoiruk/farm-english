# 🎮 Interactive Features — FarmEnglish

## Реализованные фичи

### 1. Web Speech API (Произношение) ✅

- Нажатие на слово/фразу → произношение en-GB
- Скорость 0.88x для начинающих
- Кнопка ▶ для автоматического воспроизведения всего диалога
- `speechSynthesis.cancel()` перед каждым новым словом

### 2. Quiz System (Тесты) ✅

- Множественный выбор (4 варианта)
- **Режим тренажера:** при неверном ответе вариант подсвечивается красным, но тест не блокируется — можно пробовать снова (обучение на ошибках).
- **Педагогическая обратная связь:** каждое задание сопровождается полем `expl` с объяснением грамматики/лексики.
- **Отображение заданий перевода:** для заданий типа `[TRANSLATE]` переводимая фраза (hint) отображается прямо в строке вопроса (inline) жирным шрифтом в кавычках (например: **🗣️ Переведите: "Доброе утро"**), что улучшает её видимость по сравнению с отдельным блоком подсказки ниже.
- **Динамическая мотивация:** 30 уникальных локализованных мотивирующих фраз (выбираются псевдослучайно), поддерживающих студента при ошибке.
- При 100% правильных ответов во всех заданиях → "Урок пройден! 🎉"
- Кнопка "Завершить урок" → сохранение в localStorage

### 3. Progress Tracking & Navigation ✅

- Экран прогресса полностью интегрирован в Главный экран (Hub) для быстрого доступа.
- Пошаговая навигация внутри урока (Tab Progression): кнопки `Далее: [Вкладка] ➔` плавно ведут пользователя от теории к практике.
- Единая плавающая кнопка «Назад» для возврата к Hub.
- Счётчик слов и статистика сохраняются в `localStorage`.

### 4. Language Switching ✅

- 5 языков: RU/UZ/TJ/KG/KZ
- Сохранение выбора в localStorage
- Синхронизация между страницами

### 5. Dialogue Auto-Play ✅

- Кнопка ▶ проигрывает все реплики по очереди
- Задержка между репликами: длина текста × 70ms + 900ms

## Планируемые фичи

### 6. Grammar Tables UI (Грамматические таблицы) 🔜

**Переключатель форм:**

- Три вкладки: ✅ Positive / ❌ Negative / ❓ Question
- По умолчанию активна ✅ Positive
- Каждая вкладка показывает таблицу с колонками:
  - Подлежащее (Subject)
  - Глагол (Verb form)
  - Пример (Example sentence)
  - Кириллическая транскрипция (Transcription)
  - Перевод (Translation — на выбранном языке)

**Визуальный стиль таблицы:**

- Мобильный формат — горизонтальный скролл при необходимости
- Зелёная подсветка для правила
- Чередование цвета строк для читаемости
- Кнопка 🔊 в каждой строке — произнести пример

**Культурные заметки:**

- Блок с иконкой 🌍 под таблицей
- Фон: мягкий жёлтый (#fffbe6)
- Объяснение различий между английским и родным языком

### 7. Streak (Дневная серия) 🔜

- Считать дни подряд, когда пользователь проходит хотя бы 1 урок
- Хранение в localStorage: `fe_streak`, `fe_last_date`
- Показывать streak на главной и экране прогресса
- Мотивирующие сообщения: "5 дней подряд! 💪"

### 8. Flashcard Mode 🔜

- Режим "карточки" для повторения слов
- Показать английское слово → нажать → увидеть перевод
- Кнопки "Знаю" / "Не знаю"

### 9. Fill-in-the-blank Exercises 🔜

- Интерактивное поле ввода вместо множественного выбора
- Проверка с учётом регистра и пробелов
- Подсказка после 2 неудачных попыток

### 10. Pronunciation Check 🔜

- Пользователь произносит слово → Web Speech Recognition API
- Сравнение с ожидаемым текстом
- Обратная связь: "Отлично!" или "Попробуйте ещё раз"
- **Ограничение:** Требует HTTPS и интернет

### 11. Offline Mode (PWA) 🔒

- Service Worker для кэширования HTML/JS
- Manifest.json для установки на главный экран
- Иконка приложения

### 12. 🏆 Certificate System (Сертификат) — ВАЖНАЯ ФИЧА

**Условие:** Сертификат доступен ТОЛЬКО после 100% прохождения всех уроков уровня.

#### Пользовательский сценарий (User Flow)

```
1. Завершён последний урок уровня
       ↓
2. Поздравительный баннер 🎉 (на языке пользователя)
       ↓
3. Модальное окно → ввод имени студента
   [Имя Фамилия          ]
   [🏆 Получить сертификат]
       ↓
4. window.print() → браузер скрывает сайт, показывает только сертификат
       ↓
5. Пользователь сохраняет PDF через диалог печати
```

#### Дизайн сертификата (A4 Portrait, Luxury Style)

- **Стиль:** Премиальный, тёмно-зелёный + золотой
- **Формат:** A4, 210mm × 297mm
- **Элементы:**
  - Логотип `🌿 FarmEnglish`
  - Заголовок `CERTIFICATE OF COMPLETION`
  - Имя студента (крупный serif-шрифт)
  - Название курса: `English for Seasonal Workers — CEFR A0–A1`
  - Подзаголовок урока: `"First Days on the Farm"`
  - Дата выдачи (en-GB формат: `24 April 2026`)
  - Статистика: `15 lessons · 450 words completed`
  - Декоративные рамки и орнаменты (CSS)
  - Подпись платформы

#### Техническая реализация (Zero-Budget, CSS Print)

```html
<style>
  @media print {
    body > * { display: none !important; }
    #certificate { display: block !important; }
  }
  @page { size: A4 portrait; margin: 0; }
</style>
```

```javascript
function generateCertificate(userName, levelId) {
  const levels = {
    a1: { title: 'CEFR A0–A1', name: 'First Days on the Farm', lessons: 15 },
    a2: { title: 'CEFR A2', name: 'Life in the Camp', lessons: 15 },
    b1: { title: 'CEFR B1', name: 'Solving Problems', lessons: 12 },
  };
  const date = new Date().toLocaleDateString('en-GB', {
    day: 'numeric', month: 'long', year: 'numeric'
  });
  document.getElementById('cert-name').textContent = userName;
  document.getElementById('cert-date').textContent = date;
  document.getElementById('cert-level').textContent = levels[levelId].title;
  
  // Сохранить в localStorage
  localStorage.setItem(`fe_cert_${levelId}_name`, userName);
  localStorage.setItem(`fe_cert_${levelId}_date`, date);
  localStorage.setItem(`fe_cert_${levelId}_issued`, 'true');
  
  window.print();
}
```

#### LocalStorage ключи для сертификатов

```javascript
fe_cert_a1_name   // Имя студента
fe_cert_a1_date   // Дата получения
fe_cert_a1_issued // 'true' если сертификат уже выдан
```
