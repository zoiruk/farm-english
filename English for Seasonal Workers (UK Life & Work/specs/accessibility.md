# ♿ Accessibility — FarmEnglish

## Целевая аудитория с точки зрения доступности

- Пользователи с **низкой цифровой грамотностью**
- Работа при **плохом освещении** (раннее утро / поздний вечер)
- Усталые руки после физической работы → нужны **большие кнопки**
- Экраны бюджетных Android-смартфонов (5-6 дюймов)

---

## Стандарты

### Текст

- Минимальный размер шрифта: **14px** (для контента), **11px** (для меток и транскрипций)
- Контрастность: WCAG AA (4.5:1 для текста, 3:1 для крупного)
- Поддержка кириллицы (RU/TJ/KG/KZ) и латиницы (UZ)
- Line-height: минимум **1.4** для читаемости
- **Адаптация для низкой грамотности:** Любые базовые английские слова, впервые представленные в правилах грамматики, ДОЛЖНЫ дублироваться кириллической транскрипцией. Пользователь не должен гадать, как читается слово. Транскрипция оформляется тегом `<span class="g-transcr">` (серый цвет, мелкий шрифт).

### Кнопки и интерактив

- Touch target: минимум **44×44px**
- Расстояние между кнопками: минимум **8px**
- Визуальная обратная связь при нажатии (scale/color change)
- Все кнопки имеют `cursor: pointer`

### Цвета

- Не полагаться только на цвет для передачи информации
- Правильный ответ: ✓ зелёный + текст "Правильно!"
- Неправильный ответ: ✗ красный + текст "Неверно"
- Прогресс: цветная полоса + числовой счётчик (N/15)

### Аудио

- Кнопка 🔊 для каждого слова, фразы и реплики
- Скорость речи: **0.88x** (медленнее нормальной)
- Британский акцент (en-GB)
- `speechSynthesis.cancel()` перед каждым новым воспроизведением

### Навигация и Layout

- Простая структура: максимум 2 уровня вложенности
- Навигация осуществляется через центральный хаб (`index.html`) и плавающую кнопку «Назад» для возврата в уроки
- Кнопка "Назад" всегда видна
- Текущая вкладка выделена цветом
- **Адаптивность (Responsive):** Избегать горизонтальной прокрутки. Таблицы (например, `g-table`) не должны иметь жесткого `min-width`, чтобы контент автоматически переносился и всегда помещался в ширину экрана (320px-400px). Кнопки управления (🔊) всегда должны быть в зоне видимости без скролла.
