# 📋 Product Specification — FarmEnglish

## 🎯 Миссия

Создать бесплатную веб-платформу для изучения английского языка сезонными рабочими в Великобритании (Seasonal Worker Visa). Платформа помогает рабочим выжить и адаптироваться с первого дня на ферме.

---

## 👥 Целевая аудитория

| Параметр | Значение |
|----------|----------|
| Кто | Сезонные рабочие (Seasonal Worker Visa, Tier 5) |
| Откуда | Узбекистан, Таджикистан, Кыргызстан, Казахстан |
| Возраст | 20–45 лет |
| Устройство | Смартфон (Android 90%, iOS 10%) |
| Интернет | Нестабильный / часто без Wi-Fi в поле |
| Уровень английского | Нулевой (A0) → Начальный (A1) |
| Время на обучение | 15–30 минут вечером после смены |

## 🧩 Компоненты каждого урока

### 1. Грамматика (3 формы + таблица + 10 примеров)

- Одно грамматическое правило на урок (20 тем CEFR → 15 уроков)
- **Три обязательных формы:** ✅ Утверждение / ❌ Отрицание / ❓ Вопрос
- Визуальные таблицы: Английский + **Кириллическая транскрипция** + Перевод
- Простое объяснение на родном языке (без лингвистических терминов)
- Предполагается **нулевой уровень** знаний
- Минимум 10 примеров из контекста фермы/рабочих
- **Культурные заметки**: объяснение концепций, которых нет в родном языке (артикли, to be)
- Заметка-подсказка (⚠️ note) с типичными ошибками
- **Контекст**: Все примеры из сценариев работы сезонных рабочих

### 2. Словарь (30 слов на урок)

- Английское слово + эмодзи
- **Кириллическая транскрипция**
- Перевод на 5 языков (RU/UZ/TJ/KG/KZ)
- Нажатие → произношение через Web Speech API
- Реалистичные слова из жизни рабочих

### 3. Диалоги (10 реплик на урок)

- Реальные ситуации на ферме
- Спикеры: Manager (m), Worker (w), Cashier (c), Doctor (d)
- Английский текст + **Кириллическая транскрипция** + перевод на язык пользователя
- Кнопка 🔊 для каждой реплики
- Кнопка ▶ для автоматического проигрывания

### 4. Тесты (10 заданий на урок)

- Множественный выбор (4 варианта)
- Заполнение пропусков
- Перевод фраз
- Мгновенная обратная связь (✓ зелёный / ✗ красный)
- При 100% правильных → "Урок пройден! 🎉"

### 5. Разговорник (общий)

- 150+ фраз для MVP
- 300 фраз — цель
- Категории: Работа, Здоровье, Зарплата, Магазин, Общение, SOS, Транспорт, Кэмп, Документы
- Каждая фраза на 5 языках + 🔊
