# ✅ Requirements — FarmEnglish

## Функциональные требования

### FR-01: Уроки

- [ ] Платформа должна отображать 15 уроков уровня A0-A1
- [ ] Каждый урок содержит 4 вкладки: Грамматика, Слова, Диалог, Тест
- [ ] Переключение между вкладками без перезагрузки страницы
- [ ] Кнопка «← Назад» для возврата к списку уроков

### FR-01b: Грамматика — 3 формы + таблицы

- [ ] Каждое грамматическое правило обязательно показывается в **трёх формах**: ✅ Утверждение / ❌ Отрицание / ❓ Вопрос
- [ ] Переключатель форм (вкладки) внутри вкладки «Грамматика»
- [ ] Визуальные **таблицы** для каждой формы: Subject / Verb / Example / Transcription / Translation
- [ ] Минимум **10 примеров** из контекста фермы с переводами на 5 языков
- [ ] **Культурные заметки** (блок 🌍): объяснение различий между английским и родным языком
- [ ] Кнопка 🔊 для произношения каждого примера в таблице

### FR-01c: Кириллическая транскрипция

- [ ] Все грамматические примеры содержат поле `transcr` — кириллическая транскрипция
- [ ] Все 30 слов урока содержат поле `transcr` — кириллическая транскрипция
- [ ] Все реплики диалога содержат поле `transcr` — кириллическая транскрипция
- [ ] Транскрипция отображается под английским текстом мелким шрифтом (12px, цвет #888)

### FR-02: Словари

- [ ] Каждый урок содержит **30 слов** с эмодзи, транскрипцией (русскими буквами) и переводом на 5 языков: RU, UZ, TJ, KG, KZ
- [ ] Нажатие на слово → воспроизведение произношения (Web Speech API) английского слова
- [ ] Слова отображаются в сетке 2 колонки (в мобильной версии), 3 колонки (в широкой версии/десктопе) для лучшей читаемости карточек

### FR-03: Диалоги

- [ ] Каждый урок содержит **10 реплик** диалога
- [ ] Визуальное разделение спикеров (Manager = зелёный фон, Worker = белый фон)
- [ ] Кнопка 🔊 для каждой реплики
- [ ] Кнопка ▶ для автоматического проигрывания всего диалога

### FR-04: Тесты

- [ ] Каждый урок содержит **10 заданий**
- [ ] Типы: множественный выбор (4 варианта)
- [ ] Мгновенная обратная связь: ✓ зелёный / ✗ красный
- [ ] При 100% правильных ответов → баннер "Урок пройден! 🎉"
- [ ] Кнопка "Завершить урок" сохраняет прогресс в localStorage

### FR-05: Разговорник

- [ ] **150+ фраз** с переводом на 5 языков
- [ ] Фильтр по категориям: Работа, Здоровье, Зарплата, Магазин, Общение, SOS
- [ ] Кнопка 🔊 для каждой фразы

### FR-06: Прогресс

- [ ] Отображение количества пройденных уроков
- [ ] Отображение количества изученных слов
- [ ] Прогресс-бар (N/15)
- [ ] Баллы за пройденные уроки

### FR-07: Локализация

- [ ] Интерфейс на 5 языках: RU, UZ, TJ, KG, KZ
- [ ] Переключатель языка в шапке
- [ ] Выбранный язык сохраняется в localStorage
- [ ] Язык синхронизируется между всеми страницами

### FR-08: Навигация

- [ ] Главная страница (index.html) с карточками уровней
- [ ] Кнопка "Начать" / "Продолжить" для каждого уровня
- [ ] Блокировка уровней A2, B1 до завершения предыдущего
- [ ] Кнопка "← Главная" на странице уроков

### FR-09: 🏆 Система сертификации

- [ ] Сертификат доступен **ТОЛЬКО** при 100% прохождении всех уроков уровня
- [ ] Проверка условия: `fe_a1.length >= 15` (все уроки сохранены в localStorage)
- [ ] После завершения последнего урока → поздравительный баннер 🎉 на языке пользователя
- [ ] Модальное окно с полем ввода **имени студента** для персонализации документа
- [ ] Кнопка "🏆 Получить сертификат" — запускает генерацию
- [ ] Генерация PDF через CSS Print API (`window.print()`) — без внешних библиотек
- [ ] Формат документа: **A4 портрет** (210mm × 297mm)
- [ ] Стиль: **премиальный, роскошный** — тёмно-зелёный + золотой, serif-шрифт для имени
- [ ] Содержимое: имя студента, название курса `"English for Seasonal Workers"`, уровень CEFR, дата выдачи (en-GB), статистика (15 уроков, 450 слов)
- [ ] Данные сертификата сохраняются в localStorage: `fe_cert_a1_name`, `fe_cert_a1_date`, `fe_cert_a1_issued`
- [ ] На экране "Прогресс" — кнопка "Скачать сертификат" если он уже был получен
- [ ] Работает полностью **offline** (нет внешних зависимостей)
- [ ] Поддерживает все 5 языков поздравления (RU/UZ/TJ/KG/KZ)

## Нефункциональные требования

### NFR-01: Производительность

- Время загрузки < 3 секунды на 3G
- Один HTML-файл на уровень (без дополнительных запросов)
- Нулевые внешние зависимости (JS/CSS frameworks)

### NFR-02: Офлайн

- Уроки доступны после первой загрузки (данные inline в HTML)
- Прогресс хранится в localStorage (не требует сервера)
- Web Speech API может требовать интернет для голосов (при отсутствии интернета при нажатии кнопки  🔊 на словах или фразах прекращать воспроизведение и отображать сообщение об отсутствии интернета)

### NFR-03: Безопасность

- Нет персональных данных на сервере
- Все данные локальны (localStorage)
- Нет аутентификации

### NFR-04: Доступность

- Touch target: минимум 44×44px
- Контрастность текста: WCAG AA
- Font-size: минимум 14px
- Поддержка кириллицы и латиницы

### NFR-05: Совместимость

- Chrome Android 80+ ✅
- Samsung Internet 15+ ✅
- Safari iOS 14+ ✅
- Firefox Android ⚠️ (ограниченная поддержка Web Speech API)
