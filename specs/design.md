# 🎨 Design Specification — FarmEnglish

## Стиль: Telegram-inspired Expressive Light Theme

Интерфейс вдохновлён Telegram — чистый, с сочными акцентами синего, адаптированный под светлую тему Material Design 3 и Google Fonts.

---

## 📐 Layout

### Контейнер

- `max-width: 430px` — оптимизирован под смартфон
- `margin: 0 auto` — центрирование на десктопе
- `min-height: 100vh` — полный экран
- `display: flex; flex-direction: column`

### Навигация (Hub-and-Spoke)

- **Навигация осуществляется через центральный хаб (`index.html`) и плавающую кнопку «Назад» для возврата в уроки.**
- **Главная (index.html):** Единый хаб, объединяющий выбор курса, быстрый переход к разговорнику и экрану прогресса.
- **Уроки (a1.html):** Минималистичный дизайн без лишних панелей. Плавающая кнопка «Назад» (frosted glass) в правом верхнем углу экрана возвращает пользователя в хаб. Внутри урока — табы (Grammar/Words/Dialogue/Quiz) с кнопками пошагового перехода `Далее: [Вкладка] ➔` цвета `#0075de` внизу каждой страницы.

## 🖲️ Компоненты UI

### Кнопки

- Минимальный размер: `44px × 44px` (touch target)
- Border-radius: `8px–12px`
- Активная анимация: `transform: scale(0.98)` при нажатии

### Карточки

- Background: `var(--card)` / `#fff`
- Border: `1px solid var(--border)`
- Border-radius: `12px`
- Padding: `14px–16px`

### Чипы/бейджи

- Border-radius: `99px` (pill shape)
- Padding: `3px 10px`
- Font-size: `11px`, weight: `700`

### Вкладки (Tabs / Segmented Control)

- **Контейнер (.tabs):** Мягкий серый фон (`#e8eaed`), отступы `3px`, `border-radius: 12px`, `display: flex`.
- **Неактивная вкладка (.tab):** Прозрачный фон, текст `#8e95a2`, `font-weight: 600`.
- **Активная вкладка (.tab.on):** Белый фон (`#fff`), легкая тень (`box-shadow`), цвет текста — фирменный синий (`#0075de`).
- **Почему такой дизайн:** Синий текст добавляет приложению сочный акцент (Telegram-style), но белый фон вкладки сохраняет премиальную чистоту и легкость (Clean Design), не отвлекая от учебного контента.


### Блоки-подсказки (Hints / .w-hint)

- Background: `#fef7f0` (светло-оранжевый)
- Text color: `#c05600`
- Border-radius: `10px`
- Выравнивание: `text-align: center`
- Шрифт: `13px`, `weight: 600`
- Использование: для коротких инструкций (например, 👇 Нажмите на слово)

### Задания перевода (Quiz Translate)

- Inline отображение целевой фразы (в одну строку с текстом задания)
- Text color: `var(--text)` (основной цвет)
- Font-weight: `600`
- Использование: выделение переводимого текста в вопросе (например: **🗣️ Переведите: "Доброе утро"**)


### Грамматические конструкции (внутри правил)

- Text color: `#1a1a2e` (черный)
- Font-weight: `700` (жирный)
- Использование: выделение английских слов (например, **I am**) внутри текстовых объяснений, чтобы они не сливались с транскрипцией.

### Прогресс-бар

- Height: `4px–8px`
- Border-radius: `99px`
- Анимация: `transition: width 0.5s`

### Bottom Sheet (выбор языка)

- Анимация: `fadein 0.15s`
- Закрытие: нажатие на оверлей
- Handle сверху: `36px × 4px`, `border-radius: 99px`

## 🔍 Спецификация 7 недостающих состояний (Missing States)

Для обеспечения непрерывности пользовательского опыта и предотвращения разочарования при работе с приложением (особенно в состоянии усталости после рабочей смены), явно специфицированы следующие состояния:

### 1. Текущий прогресс в тесте (Quiz Progress Counter)
- **Компонент**: Заголовок карточки вопросов или верхняя панель теста.
- **Поведение**: Отображает текущий статус прохождения в формате `Вопрос X из 10` (например, "Вопрос 3 из 10").
- **Стиль**: Цвет `var(--text3)` или `var(--c-accent)`, шрифт `14px`, полужирный. Обновляется динамически при переходе к следующему вопросу или прокрутке.

### 2. Состояние "Ответил на все вопросы, но есть ошибки" (Quiz Validation Alert)
- **Компонент**: Предупреждающий блок перед кнопкой завершения теста.
- **Условие**: Пользователь заполнил все 10 полей ответов, но некоторые из них неверны. Кнопка "Завершить" остается заблокированной (`:disabled`).
- **Визуальное отображение**: 
  - Отображается информационный баннер цвета `#fdf2f2` (мягкий красный) с красной рамкой.
  - Текст ошибки: *"Пожалуйста, исправьте выделенные красным ответы, чтобы завершить тест."* (локализовано на 5 языков).
  - Элементы с неверными ответами подсвечиваются красной границей и содержат иконку ❌.

### 3. Сбой синтеза речи / Недоступность аудио (Speech API Fallback)
- **Компонент**: Кнопки воспроизведения аудио 🔊.
- **Условие**: Устройство не поддерживает Web Speech API, нет интернет-подключения для загрузки голосов (на некоторых версиях Android) или громкость отключена.
- **Поведение**: При попытке нажатия воспроизводится всплывающее уведомление (Toast): *"Аудио временно недоступно. Используйте транскрипцию."* (локализовано). На кнопках 🔊 отображается визуальный индикатор отключения звука (emoji 🔇 или перечеркнутая иконка).

### 4. Первое посещение / Пустое состояние (First-time Onboarding Hub)
- **Компонент**: Главный хаб (`index.html`) при отсутствии прогресса в `localStorage`.
- **Поведение**: Вместо пустых нулей статистики "0 уроков / 0 слов" выводится карточка онбординга:
  - **Заголовок**: *"Добро пожаловать в FarmEnglish! 👋"*
  - **Текст**: *"Начните с первого урока A1, чтобы освоить базовый английский язык для жизни и работы на фермах Великобритании. Ваш прогресс сохраняется автоматически."*
  - **CTA кнопка**: *"Начать Урок 1"* (перенаправляет на `a1.html?lesson=1`).

### 5. Сбой localStorage / Режим инкогнито (Storage Exception Fallback)
- **Компонент**: Информационный тост/баннер на хабе и страницах уроков.
- **Условие**: Браузер в режиме инкогнито или заполнена квота (`QuotaExceededError`).
- **Поведение**: Приложение работает в памяти (in-memory fallback), но выводит неблокирующее предупреждение: *"Режим инкогнито: ваш прогресс не будет сохранен после закрытия вкладки."* При ошибке квоты выводится: *"Не удалось сохранить прогресс. Освободите место на устройстве."*

### 6. Ошибка генерации сертификата (Certificate Error State)
- **Компонент**: Модальное окно сертификата.
- **Условие**: Имя пользователя не введено или содержит некорректные символы, либо в localStorage нет записи о завершении всех 15 уроков.
- **Поведение**: Кнопка "Создать сертификат" блокируется, под полем ввода выводится текст ошибки: *"Пожалуйста, введите ваше имя на латинице"* красным цветом. Если уроки не пройдены, модальное окно не открывается, а выводится предупреждение.

### 7. Загрузка шрифта иконок (Woff2 Font Loading Fallback)
- **Поведение**: При медленном 3G-соединении, пока загружается файл шрифта `.woff2`, вместо пустых мест на экране отображаются стандартные текстовые эквиваленты или Emoji (например, `[Back]` вместо стрелки, `[Home]` вместо 🏠), предотвращая невидимый текст (FOIT) с помощью директивы `font-display: swap` и резервных CSS-стилей.
