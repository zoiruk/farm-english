# 🎨 Design Specification — FarmEnglish

## Стиль: Telegram Dark Theme

Интерфейс вдохновлён Telegram — чистый, с акцентами синего.  + Material Design 3, Google Fonts

## 📐 Layout

### Контейнер

- `max-width: 430px` — оптимизирован под смартфон
- `margin: 0 auto` — центрирование на десктопе
- `min-height: 100vh` — полный экран
- `display: flex; flex-direction: column`

### Навигация (Hub-and-Spoke)

- **Навигация осуществляется через центральный хаб (`index.html`) и плавающую кнопку «Назад» для возврата в уроки.**
- **Главная (index.html):** Единый хаб, объединяющий выбор курса, быстрый переход к разговорнику и экрану прогресса.
- **Уроки (a1.html):** Минималистичный дизайн без лишних панелей. Плавающая кнопка «Назад» (frosted glass) в правом верхнем углу экрана возвращает пользователя в хаб. Внутри урока — табы (Grammar/Words/Dialogue/Quiz) с кнопками пошагового перехода `Далее: [Вкладка] ➔` цвета `#0075de` внизу каждой страницы.

## 🖲️ Компоненты UI

### Кнопки

- Минимальный размер: `44px × 44px` (touch target)
- Border-radius: `8px–12px`
- Активная анимация: `transform: scale(0.98)` при нажатии

### Карточки

- Background: `var(--card)` / `#fff`
- Border: `1px solid var(--border)`
- Border-radius: `12px`
- Padding: `14px–16px`

### Чипы/бейджи

- Border-radius: `99px` (pill shape)
- Padding: `3px 10px`
- Font-size: `11px`, weight: `700`

### Вкладки (Tabs / Segmented Control)

- **Контейнер (.tabs):** Мягкий серый фон (`#e8eaed`), отступы `3px`, `border-radius: 12px`, `display: flex`.
- **Неактивная вкладка (.tab):** Прозрачный фон, текст `#8e95a2`, `font-weight: 600`.
- **Активная вкладка (.tab.on):** Белый фон (`#fff`), легкая тень (`box-shadow`), цвет текста — фирменный синий (`#0075de`).
- **Почему такой дизайн:** Синий текст добавляет приложению сочный акцент (Telegram-style), но белый фон вкладки сохраняет премиальную чистоту и легкость (Clean Design), не отвлекая от учебного контента.


### Блоки-подсказки (Hints / .w-hint)

- Background: `#fef7f0` (светло-оранжевый)
- Text color: `#c05600`
- Border-radius: `10px`
- Выравнивание: `text-align: center`
- Шрифт: `13px`, `weight: 600`
- Использование: для коротких инструкций (например, 👇 Нажмите на слово)

### Задания перевода (Quiz Translate)

- Inline отображение целевой фразы (в одну строку с текстом задания)
- Text color: `var(--text)` (основной цвет)
- Font-weight: `600`
- Использование: выделение переводимого текста в вопросе (например: **🗣️ Переведите: "Доброе утро"**)


### Грамматические конструкции (внутри правил)

- Text color: `#1a1a2e` (черный)
- Font-weight: `700` (жирный)
- Использование: выделение английских слов (например, **I am**) внутри текстовых объяснений, чтобы они не сливались с транскрипцией.

### Прогресс-бар

- Height: `4px–8px`
- Border-radius: `99px`
- Анимация: `transition: width 0.5s`

### Bottom Sheet (выбор языка)

- Анимация: `fadein 0.15s`
- Закрытие: нажатие на оверлей
- Handle сверху: `36px × 4px`, `border-radius: 99px`

## 📱 Адаптивность

| Устройство | Поведение |
|-----------|-----------|
| < 360px | Шрифт чуть меньше, padding 12px |
| 360–430px | Основной дизайн |
| > 430px | Центрирование, max-width 430px |

Для словаря используется более спокойная сетка: **2 колонки на обычном мобильном экране** и **3 колонки на широком экране/десктопе**. Это сохраняет крупные карточки и снижает визуальную тесноту.

## 🎭 Иконки

- **В основном Emoji** — для скорости загрузки и нулевых зависимостей.
- Исключение: плавающая кнопка «Назад» использует минималистичную SVG-иконку шеврона (в стиле iOS) для премиального вида.
- Эмодзи для навигации: 🏠 💬 ⚙️
- Эмодзи для категорий фраз: 🔊 👷 🏥 💷 🛒 🤝 🆘
- Эмодзи для слов: 🍓 🚜 🧤 🥾 ☕ и т.д.
