# FarmEnglish �

**Бесплатный английский для сезонных рабочих в Великобритании**

Веб-платформа для изучения английского языка сезонными рабочими (Seasonal Worker Visa). Помогает выжить и адаптироваться с первого дня на ферме — без интернета, без регистрации, без оплаты.

🔗 **[Открыть платформу](https://zoiruk.github.io/farm-english/)**

---

## Для кого

Сезонные рабочие из Узбекистана, Таджикистана, Кыргызстана, Казахстана. Уровень английского: нулевой (A0) → начальный (A1). Устройство: смартфон Android.

## Что внутри

| Уровень | Уроков | Статус |
|---------|--------|--------|
| A0–A1 «Первые дни на ферме» | 15 | ✅ Готово |
| A2 «Жизнь в кэмпе» | 15 | 🔜 В разработке |
| B1 «Решение проблем» | 12 | 🔜 Планируется |

Каждый урок: грамматика (3 формы + таблицы), 30 слов с кириллической транскрипцией, диалог из 10 реплик, 10 тестов. Интерфейс на 5 языках: 🇷🇺 🇺🇿 🇹🇯 🇰🇬 🇰🇿

## Для разработчиков

### Стек

- Vanilla HTML5 + CSS3 + JavaScript (ES6+) — без фреймворков, без сборщика
- Web Speech API (`en-GB`, rate 0.88) — произношение
- localStorage — прогресс пользователя
- GitHub Pages — хостинг

### Структура

```
index.html          ← Главная: выбор уровня, прогресс, разговорник
a1.html             ← A0–A1 курс (15 уроков, все данные inline)
translations.js     ← i18n для index.html (5 языков)
scratch/            ← Скрипты аудита и сборки (Node.js)
specs/              ← Документация продукта
```

### Запуск аудита перед коммитом

```bash
node scratch/audit.js && node scratch/advanced_audit.js
```

Урок считается готовым только при нулевых ошибках в обоих аудитах.

### Добавление нового уровня

1. Скопировать `a1.html` → `a2.html`
2. Изменить `--c-accent: var(--c-a2)` в `:root`
3. Заполнить массив `LESSONS` по формату из `specs/tech.md`
4. Пройти аудит
5. Добавить карточку уровня в `index.html`
